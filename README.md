# 🪑 智能班级座位表生成工具

一个功能丰富的班级座位表在线生成与管理工具，帮助老师快速、科学地安排学生座位。

> 📎 **在线访问**: [https://liao-jw.github.io/class-seating-chart/](https://liao-jw.github.io/class-seating-chart/)

---

## ✨ 功能特性

### 📁 学生名单管理
- **多格式导入**：支持 Excel（.xlsx）、CSV、TXT、JSON 格式的学生名单文件
- **模板下载**：提供标准学生名单模板，包含姓名、性别、身高、成绩、视力等字段
- **拖拽上传**：直接拖拽文件到上传区域即可导入

### 🪑 多种排座算法
- **随机排座**：完全随机打乱学生顺序
- **按身高排序**：从低到高安排座位，方便教室视线管理
- **按成绩均衡分配**：将不同成绩层次的学生均匀分布到各区域
- **按性别穿插排列**：男女学生交替就座
- **多因素智能排座**：综合考虑身高、视力、成绩、性别四个因素，支持自定义各因素权重

### 📐 灵活的座位布局
- 自定义行数（3-15行）和列数（3-15列）
- 自适应讲台显示
- 支持撤销/重做操作，最多记录50步历史

### 🔒 座位约束系统
- **必须坐在一起**：指定某些学生必须相邻就座
- **不能坐在一起**：指定某些学生必须分开就座
- **约束冲突检查**：一键检测当前座位安排是否存在约束违反
- **座位锁定**：双击学生座位可锁定/解锁，锁定后的座位在重新生成时保持不变

### 🎯 批量操作
- **批量选择模式**：同时选中多个座位进行操作
- **批量移动**：将选中学生一次性移动到新位置
- **全向轮换**：支持左移、右移、上移、下移四种方向的座位整体轮换

### 🏷️ 学生标签系统
内置7种学生标签，方便老师标记学生特殊属性：
- 近视、远视、听力障碍
- 优等生、需关注、班干部、特长生
- 支持为每个学生添加自定义备注

### 💾 数据保存与导出
- **本地保存**：座位表数据自动保存到浏览器本地存储
- **历史记录**：保存多版座位表方案，随时切换回顾
- **多格式导出**：
  - Excel 文件（.xlsx）
  - 图片（.png）
  - PDF / 打印

## 📖 使用方法

1. 打开 [在线地址](https://liao-jw.github.io/class-seating-chart/)
2. 下载学生名单模板，填写学生信息后上传（或直接导入已有名单文件）
3. 设置座位行列数，选择排座规则
4. 点击「生成座位表」，系统自动安排座位
5. 可手动拖拽调整个别座位，设置约束条件
6. 导出为 Excel、图片或 PDF
